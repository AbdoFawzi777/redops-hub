What it does
Everything a Red Team needs — in one app
Built for operators who can't afford to be tied to a laptop during an active engagement.
📡
C2 Dashboard
Monitor active agents and dispatch quick commands from mobile. Get real-time beacon alerts via WebSocket — no laptop required.
🐛
Vulnerability Tracker
Log findings instantly with severity, PoC, and CVE. Share with developers in real time — no more waiting for end-of-engagement reports.
⌨
Payload Vault
AES-256 encrypted library of scripts and cheat sheets. Works fully offline. One tap copies to clipboard — ready for immediate use in the field.
🎤
Field Reporter
Write intelligence notes mid-operation. Voice-to-text and AI classification coming in v2.0 — turns spoken findings into structured vulnerability tickets.
🛡
Dev Playbooks
Secure code remediation guides in Dart, Python, and Node.js. Developers get the exact fix alongside the finding — no back-and-forth needed.
👁
Lockdown System
4-digit PIN + biometric gate on every session. Progressive lockout after 3 failed attempts. Auto-lock on background. Your data stays yours.
Security model
Built for hostile environments
Every design decision assumes the device could be seized. Sensitive data stays protected even in the worst case.
🔒
AES-256 at rest
All vulnerability data and payloads encrypted locally. Keys stored in Android hardware-backed Keystore — never in plain text.
👁
Biometric gate
Fingerprint or PIN required every session. Progressive lockout with 3 failed attempts. No bypass without device credentials.
📵
Screenshot prevention
FLAG_SECURE blocks screen capture and recent-apps thumbnails. Nothing leaks from the task switcher.
📶
Offline-first
Payload Vault works with zero connectivity. No telemetry, no analytics, no SDKs phoning home. Your operations stay private.
🛡
Code obfuscation
Release builds use ProGuard and Dart obfuscation. Reverse engineering the APK reveals nothing useful about internal logic.
📜
Certificate pinning
All API connections pin the server certificate. MITM attacks against C2 traffic are blocked at the transport layer.
